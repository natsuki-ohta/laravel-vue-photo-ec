## サイトの目的

Laravel と Vue を用いた、実際に使われているような EC サイト構成を意識したサイトです。
ポートフォリオとしての側面と、実際に自分で撮影した写真やその活動を紹介するためという側面もあります。

- 管理側・利用者側の責務分離
- API設計
- 実装後の改修を前提とした構成

を意識しております。
---

## 主な機能

### ユーザー側
- 商品一覧
- カート機能
- 注文処理

### 管理者側
- 商品管理（新規登録・編集・削除）
- 画像アップロード
- 注文情報の確認
